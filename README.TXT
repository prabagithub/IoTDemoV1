# HOW TO RUN

# Connect to "http://iotapp.cfapps.io" via Chrome Browser

# User Provisioning:
	1. Provide Your Mobile Number; Select a PLAN; Click "Submit" Button
	2. See a browser Pop-Up - Displays - Congratulations Message

# Mock Data
	3. Provide Your Mobile Number 
	4. Location; Data Category; Data Volume
	5. Click "Save" Button
	6. Observe your Bill Details appear in the Current Bill Details Table
	
# View Current Bill
	7. Provide Your Mobile Number
	8. Click "Show Bill"; 
	9. Observe your Bill Details appear in the Current Bill Details Table
